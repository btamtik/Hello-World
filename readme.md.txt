#i200 Projekt

Projekti kohta on olemas ainult algus. Kuna kogemus on 0 siis ei oska ka tulemust veel ette kujutada. 
Projekti alguseks on memory mäng.
Mängu teen väikesele lapsele ja plaan on mängus kasutada temale tuttavate loomade pilte.
Pildipaari leidmisel tuleb ekraanile ka looma nimi (õpib tähti ja kirjapilti)
Lisaks õige paari puhul lisaks nt plaksutava looma koos heliga, et lapsel ikka põnev oleks.
Kuna see pidi liiga kerge olema siis see on kõigest algus ja edasime on kõigile üllatus.
